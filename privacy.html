clipfactory — Privacy Policy

Last updated: 7 July 2026 · Canonical URL: https://aizenscode.github.io/clipfactory-site/privacy.html

clipfactory is a personal, single-user command-line application. It is operated by, and used exclusively by, the owner of the YouTube channel @surfs_tories (the "operator") to upload the operator's own original videos to the operator's own YouTube channel. It is not distributed, sold, or made available to any other person, and it has no public interface.

Use of YouTube API Services

clipfactory uses YouTube API Services. By authorizing the application, the operator agrees to be bound by the YouTube Terms of Service. Google's handling of data is described in the Google Privacy Policy.

What data the application accesses and stores

  • OAuth 2.0 credentials for the operator's own Google account (scope: youtube.upload), stored locally on the operator's personal computer and used solely to upload videos to the operator's own channel.
  • Video IDs and timestamps of the operator's own uploads, stored locally in a state file to prevent duplicate uploads.

The application does not access, collect, or store any data belonging to any person other than the operator. It does not read comments, subscriber lists, analytics about other users, or any third-party content.

Data sharing

No data obtained through YouTube API Services is shared with, sold to, or disclosed to any third party. All stored data remains on the operator's personal computer.

Data retention and deletion

Stored credentials and upload records are kept only as long as the application is in use and can be deleted at any time by removing the local files. No copies exist elsewhere.

Cookies and tracking

The application is a command-line tool and this is a static page: neither uses cookies or any tracking technology.

Revoking access

Access granted to clipfactory can be revoked at any time via Google's security settings at https://myaccount.google.com/permissions.

Contact

For any question about this policy, contact the operator via the YouTube channel @surfs_tories.
